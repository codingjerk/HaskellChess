displayPiece :: Piece -> String

This function gets a piece and returns it's string representation for displaing on Board.

pieceToFen :: Piece -> String

pieceToFen function gets a piece and returns it's string representation in FEN-notation. It is just one char.

pieceFromFen :: Char -> Piece

pieceFromFen function gets a character and returns Piece, what represent it in FEN-notation.

Uppercase characters is White pieces. Lowercase characters is Black pieces.

Also, that function may generate error, if character isn't FEN-piece.

pieceTypeToFen :: PieceType -> String

pieceTypeToFen just gets PieceType and returns uppercase character, that implements it in FEN-notation.
